# Middle-end

Program transformations between parsing and backend emission.

Key directories:
- `automaton_core/` generic automaton algorithms.
- `logic/` FO/LTL logic utilities.
- `monitor_generation/` build monitor automaton from formulas.
- `contracts/` user contract coherency.
- `monitor_instrument/` inject monitor constraints into transitions.
- `stages/` orchestrates middle-end stages.
