Roadmap

Semantic Core

  • SpecV2.ReactiveModel defines reactive programs and their concrete executions.
  • SpecV2.ConditionalSafety defines safety automata, conditional specifications, and the semantic predicates AvoidA and AvoidG.
  • SpecV2.ExplicitProduct defines the explicit product and dangerous product steps.

Local Proof Layer

  • SpecV2.GeneratedClauses defines the semantic clauses extracted from the product.
  • SpecV2.RelationalTriples packages these clauses into relational Hoare triples over ticks.

Meta-Theory

  • SpecV2.Soundness proves the main soundness theorem.
  • SpecV2.RelativeCompleteness proves the relative-completeness directions.

Concrete Example

  • SpecV2.ResettableDelayExample instantiates the framework on the resettable-delay running example.
