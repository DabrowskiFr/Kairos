Soundness

SpecV2.Soundness proves that valid generated triples, together with true user invariants on admissible executions, imply conditional safety.

Relative Completeness

SpecV2.RelativeCompleteness proves the converse admissible-run results:

  • global correctness implies admissible validity of generated safety triples;
  • under true invariants on admissible runs, global correctness implies admissible validity of all generated triples.
