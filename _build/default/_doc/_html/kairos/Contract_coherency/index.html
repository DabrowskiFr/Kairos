Contract Coherency

val ensure_next_requires : Ast.node -> Ast.node

Build state invariants from user ensures and inject:

  • requires Inv(src) on each transition
  • ensures Shift(Inv(dst)) on each transition Also records these invariants in node.attrs.invariants_state_rel and adds an initialization coherency goal for init_state.
val validate_user_pre_k_definedness : ?monitor_automaton:Automaton_engine.automaton -> Ast.node -> unit

Reject contracts that reference pre_k before it can be defined from init_state. If a monitor automaton is provided, reachability is computed on the product (program states, monitor states).

val user_contracts_coherency : Ast.node -> Ast.node

Backward-compatible entry point: same transformation as ensure_next_requires.
